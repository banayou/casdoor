## 🔧 常见任务

### 如何开始学习？

1. 阅读 [Casdoor 学习指南（精简版）](./Casdoor中文学习文档-精简版.md)
2. 按照快速开始章节部署 Casdoor
3. 理解核心概念（组织、用户、应用、角色）
4. 尝试集成到你的应用

### 如何查找 API？

1. 查看 [Casdoor API 参考文档](./Casdoor-API参考文档.md)
2. 找到对应的 API 分类（认证、用户、应用等）
3. 查看详细的参数说明和示例
4. 参考 SDK 示例代码

### 如何部署到生产环境？

1. 参考学习指南中的 [部署指南](./Casdoor中文学习文档-精简版.md#8-部署指南)
2. 选择合适的部署方式（Docker、Kubernetes、宝塔面板）
3. 配置 SSL 证书和反向代理
4. 遵循安全最佳实践
